# 马来语姓名中文音译指南

> **核心原则**：马来西亚华文媒体（星洲日报、南洋商报、联合日报等）对马来语人名有约定俗成的中文音译标准，与大陆新华社音译规范**显著不同**。本项目数据面向亚太区域研究，必须使用马来西亚本地华文媒体标准译名。

## 2. 判定标准：用哪个译法？

**判定规则（按优先级排列）：**

1. **人物本人或其官方社交媒体使用的中文译名** → 最高优先级
   - 例如：Mustapha Sakmud 的 Facebook 使用"慕斯达法沙末" → 使用"慕斯达法沙末"
   - 例如：Sultan Ibrahim 的官方中文文告使用"苏丹依布拉欣" → 使用"苏丹依布拉欣"

2. **马来西亚华文主流媒体的一致用法** → 次高优先级
   - 参考媒体：星洲日报 (sinchew.com.my)、南洋商报 (nanyang.com.my)、联合日报
   - 例如：搜索 "Hamzah Zainudin site:sinchew.com.my" 确认译名

3. **中文维基百科（zh.wikipedia.org）的条目标题** → 第三优先级
   - 中文维基通常遵循马来西亚华文媒体标准
   - 例如："哈姆扎·扎因丁" 是大陆机器人批量创建的译名，应以马来西亚华文标准"韩沙再努丁"替代

4. **大陆新华社译法** → **不得使用**（除非该人物在中国官方报道中有固定译名且与马华标准一致）

## 3. key_people 命名格式

**标准格式**：`中文名 (English name)`

```
✅ "阿末扎希·哈米迪 (Ahmad Zahid Hamidi)"
✅ "韩沙再努丁 (Hamzah Zainudin)"
✅ "苏丹依布拉欣·依斯迈 (Sultan Ibrahim Ismail ibni Almarhum Sultan Iskandar)"
✅ "刘强燕 (Alice Lau)"

❌ "Ahmad Zahid Hamidi (阿末扎希·哈米迪)"  ← 反了
❌ "Hamzah Zainudin (韩沙·再努丁)"  ← 反了 + 多余分隔号
```

**华族人物**：中文姓名在前，无需中括号
- ✅ "刘强燕 (Alice Lau)"
- ✅ "魏家祥 (Wee Ka Siong)"

**马来/印度族人物**：中文译名在前
- ✅ "安华·依布拉欣 (Anwar Ibrahim)"
- ✅ "哥宾星 (Gobind Singh Deo)"

**分隔号使用**：
- 中文译名中音节边界使用 `·`（间隔号）
- "阿末扎希·哈米迪"（名·姓之间）
- "韩沙再努丁"（同一部分名不加分隔号，"韩沙"和"再努丁"紧凑连写）

## 5. 验证方法

### 5.1 搜索验证

对每个马来/印度族人物的中文名，执行以下验证：

```
搜索1: "{person_name_en} site:sinchew.com.my" — 星洲日报
搜索2: "{person_name_en} site:nanyang.com.my" — 南洋商报
搜索3: "{person_name_en} 中文名" — 通用搜索
```

### 5.2 官方来源验证

```
搜索4: "{person_name_en} site:facebook.com" — 查看其本人社交媒体使用的中文译名
搜索5: "{person_name_en} site:gov.my" — 马来西亚政府官方文件
```

### 5.3 Wikipedia 交叉验证

- 检查中文维基百科条目标题（通常是马来西亚华文标准）
- 但注意：维基数据（Wikidata）的中文标签可能由大陆机器人批量生成，**不一定是马来西亚华文标准**
- 如果中文维基和 Wikidata 中文标签不一致，以中文维基为准
